# Analisis Sentimen Wisata Banda Aceh

Penelitian skripsi mengenai analisis sentimen persepsi wisatawan terhadap destinasi wisata di Banda Aceh berdasarkan ulasan pada platform digital.

## Sumber Data
- Google Maps
- TikTok

## Dataset
- 30 destinasi wisata
- 2270 ulasan Google Maps
- 466 komentar TikTok
- Total 2736 data

## Tools
- Python
- Google Colab
- Pandas
- Sastrawi
- Scikit-Learn
- Laravel
